# Phaser Game Code

This directory contains the Phaser game logic and components for your Next.js application.

## Directory Structure

- `scenes/` - Phaser scene classes (Preload, MainMenu, Game, etc.)
- `components/` - React components that integrate with Phaser
- `utils/` - Helper functions and utilities for the game
- `types/` - TypeScript type definitions for game objects

## Integration with Next.js

The Phaser game can be integrated into your Next.js pages using React components. See the `components/` folder for examples of how to properly mount and manage Phaser instances within React.
